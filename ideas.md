## Idea A — "Chalkboard Classroom" (Nostalgic Pedagogy)

**Design Movement:** Neo-brutalist meets retro-educational
**Core Principles:**
- Dark green chalkboard texture as the primary canvas
- Handwritten-style typography for questions
- Tactile, physical feel — like writing on paper
- High contrast chalk-white text on dark backgrounds

**Color Philosophy:** Deep forest green (#1a3a2a) backgrounds evoking a real chalkboard; chalk-white (#f5f0e8) text; yellow (#f5c842) accent for highlights and correct answers; red (#e05252) for wrong answers. Emotional intent: nostalgia, focus, academic seriousness.

**Layout Paradigm:** Single-column centered card layout with generous vertical rhythm. Each question lives in its own "chalkboard panel." No sidebar.

**Signature Elements:**
- Subtle chalk-dust texture overlay on cards
- Underline inputs that look like lines on notebook paper
- "Eraser smudge" animation on wrong answers

**Interaction Philosophy:** Immediate feedback with animated chalk-writing effect for correct answers.

**Animation:** Slide-in from bottom for new questions; shake + smudge for wrong; bounce + glow for correct.

**Typography System:** "Caveat" (Google Fonts) for question text; "Space Mono" for numbers and equations; system sans for UI labels.

## Idea B — "Scientific Notebook" (Clean Academic) ✅ CHOSEN

**Design Movement:** Swiss International Typographic Style meets modern SaaS
**Core Principles:**
- Crisp white/off-white background with strong typographic hierarchy
- Mathematical notation displayed with monospace elegance
- Clear visual separation between question, input, and feedback zones
- Progress and score tracking as first-class UI elements

**Color Philosophy:** Off-white (#fafaf8) background; deep navy (#1e2d4a) for headings; slate blue (#4a6fa5) as primary interactive color; emerald (#22c55e) for correct; rose (#ef4444) for incorrect; amber (#f59e0b) for hints. Emotional intent: clarity, intelligence, trust, encouragement.

**Layout Paradigm:** Asymmetric two-zone layout: left sidebar with category selector + score tracker; right main area with question card. On mobile, collapses to full-width stacked.

**Signature Elements:**
- Thin ruled lines as decorative separators (like graph paper)
- Monospace equation display with subtle background highlight
- Animated progress bar that fills with each correct answer

**Interaction Philosophy:** Keyboard-first — press Enter to submit, Tab to navigate. Instant visual feedback with smooth color transitions.

**Animation:** Fade + slide for question transitions; spring bounce on score increment; subtle pulse on input focus.

**Typography System:** "DM Serif Display" for the site title; "Source Serif 4" for question text; "JetBrains Mono" for equations and numbers; "DM Sans" for UI labels and buttons.

## Idea C — "Neon Arcade" (Gamified Learning)

**Design Movement:** Cyberpunk / Retro-arcade gamification
**Core Principles:**
- Dark background with neon accent colors
- Score, streak, and timer as central game mechanics
- Each question category has its own neon color identity
- Pixel-art inspired borders and decorative elements

**Color Philosophy:** Near-black (#0d0d1a) base; electric blue (#00d4ff) for linear equations; hot pink (#ff2d78) for combinatorics; gold (#ffd700) for score; green (#39ff14) for correct. Emotional intent: excitement, competition, reward.

**Layout Paradigm:** Full-screen game-like layout with a HUD (heads-up display) at the top showing score/streak/timer, and a centered question arena below.

**Signature Elements:**
- Glowing neon borders on cards
- Pixel-font for score display
- Particle explosion effect on correct answers

**Interaction Philosophy:** Gamified — streaks, combo multipliers, time pressure optional mode.

**Animation:** Neon flicker on load; explosion particles on correct; screen shake on wrong; score counter rolls up.

**Typography System:** "Press Start 2P" for score/HUD; "Orbitron" for category labels; "Rajdhani" for question body text.
